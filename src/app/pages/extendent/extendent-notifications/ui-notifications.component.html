Bootstrap Toasts

Default Toast

Use toast class to set a default toast.

@if(show){ Symox 06 mins ago Hello, world! This is a toast message. }
Translucent

Toasts are slightly translucent, too, so they blend over whatever they might appear over.

@if(translucent){ Symox 11 mins ago Hello, world! This is a toast message. }
Stacking

When you have multiple toasts, we default to vertically stacking them in a readable manner.

@if(stacking){ Symox Just now See? Just like this. } @if(stackingsecond){ Symox 2 seconds ago Heads up, toasts will stack automatically }
Placement
